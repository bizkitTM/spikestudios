# Spike Studios

Marketing site for [spikestudios.com](https://spikestudios.com).

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Hosted on Vercel

## Develop

```bash
npm install
npm run dev
# http://localhost:3000
```

## Deploy

Push to `bizkitTM/spikestudios` on GitHub. Vercel auto-deploys on `main`.

DNS is managed in Network Solutions:
- `A @ → 76.76.21.21`
- `CNAME www → cname.vercel-dns.com`
